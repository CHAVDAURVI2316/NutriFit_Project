# 🥗 NutriFit – Nutrition & Fitness Web Application

## 📌 Project Overview
NutriFit is a web-based application designed to promote healthy living by providing users with nutrition guidance and fitness support. The platform allows users to manage their profiles and book appointments with nutrition experts for personalized diet plans.

This project demonstrates full-stack web development skills with a focus on user-friendly design and real-world functionality.

---

## 🚀 Features

- 👤 User Registration & Login  
- 📝 Profile Management  
- 📅 Appointment Booking System  
- 📞 Contact & Inquiry Form  
- 🥗 Nutrition & Diet Information  
- 📱 Responsive Design  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Django)  
- **Database:** SQLite / MySQL

## 💻 Installation & Setup

1. Clone the repository:
git clone https://github.com/YOUR-USERNAME/NutriFit.git

2. Navigate to project folder:
cd NutriFit

3. Install dependencies:
pip install django

4. Run migrations:
python manage.py migrate

5. Start server:
python manage.py runserver

6. Open in browser:
http://127.0.0.1:8000/
